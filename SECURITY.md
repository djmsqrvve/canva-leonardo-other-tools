# Security Policy

## Supported Surface
Security fixes are scoped to the active runtime:
- `dj_msqrvve_brand_system/`
- `dashboard/`

Archived scaffolds and historical docs are not actively maintained.

## Reporting
Please report security issues through GitHub private vulnerability reporting or by opening a private security advisory for this repository.

Do not post credential leaks, OAuth tokens, or exploitable details in public issues.
